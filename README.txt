=== mplabs v1.1===

Multiphase Lattice Boltzmann Suite. Released under GPLv3 (see file COPYING.txt).


=== INSTALLATION ===

A full installation requires working versions of:

 * gfortran
 * make

To install depSolver Type:

$ ./install.sh

This will compile all the binary files and move them to the mplabs/bin/ directory.
